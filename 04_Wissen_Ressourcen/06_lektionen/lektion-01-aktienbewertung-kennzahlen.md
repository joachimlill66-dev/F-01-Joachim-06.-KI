# 1. 🏗️ Die Bilanz — Das Fundament eines Unternehmens

> **Die Metapher:** Die Bilanz ist wie ein **Foto** von allem, was ein Unternehmen besitzt und schuldet — aufgenommen an einem bestimmten Stichtag. Es zeigt dir: Wie solide steht der Laden?

Die Bilanz hat zwei Seiten, die sich immer die Waage halten:

```
┌─────────────────────────────────────────────────┐
│                    BILANZ                        │
├────────────────────┬────────────────────────────┤
│   AKTIVA (links)   │   PASSIVA (rechts)         │
│   = Was das Unter- │   = Woher das Geld kam     │
│     nehmen BESITZT │                             │
│                    │   ┌──────────────────────┐  │
│ • Gebäude          │   │ Eigenkapital (EK)    │  │
│ • Maschinen        │   │ = Geld der Eigentümer│  │
│ • Vorräte          │   └──────────────────────┘  │
│ • Bargeld          │   ┌──────────────────────┐  │
│ • Forderungen      │   │ Fremdkapital (FK)    │  │
│                    │   │ = Schulden/Kredite    │  │
│                    │   └──────────────────────┘  │
├────────────────────┼────────────────────────────┤
│ Summe Aktiva       │   Summe Passiva            │
│        = BILANZSUMME (immer gleich!)             │
└─────────────────────────────────────────────────┘
```

**Die goldene Regel:** Aktiva = Passiva. Immer. Ohne Ausnahme.

---

## 1.1 Eigenkapital (Equity)

> **Was ist das?** Das Geld, das wirklich den Eigentümern (= dir als Aktionär!) gehört. Es ist das, was übrig bleibt, wenn du alle Schulden abziehst.

**Formel:**
```
Eigenkapital = Gesamtvermögen (Aktiva) − Schulden (Fremdkapital)
```

**Beispiel:**
Ein Unternehmen besitzt insgesamt 100 Mio. € an Vermögenswerten und hat 60 Mio. € Schulden.
→ **Eigenkapital = 100 − 60 = 40 Mio. €**

**Was sagt dir das?**
- Hohes Eigenkapital = Unternehmen steht auf **stabilem Fundament**
- Steigendes Eigenkapital über die Jahre = das Unternehmen wird **immer wertvoller**
- Negatives Eigenkapital = 🚨 **Alarmstufe Rot** — die Schulden übersteigen den Besitz

**Worauf achten:**
- Eigenkapital sollte über die Jahre **wachsen** (nicht schrumpfen)
- Manche Unternehmen haben gezielt niedriges EK, weil sie Aktien zurückkaufen (z. B. Apple) — das ist nicht automatisch schlecht, aber du musst es verstehen

---

## 1.2 Bilanzsumme (Total Assets)

> **Was ist das?** Die Summe von allem, was das Unternehmen besitzt — Gebäude, Maschinen, Bargeld, Patente, Vorräte, Forderungen an Kunden… alles zusammen.

**Was sagt dir das?**
- Gibt dir eine Vorstellung von der **Größe** des Unternehmens
- Alleine wenig aussagekräftig, aber wichtig als **Bezugsgröße** für andere Kennzahlen

---

## 1.3 Eigenkapitalquote

> **Die Metapher:** Stell dir vor, du kaufst ein Haus für 300.000 €. Wenn du 150.000 € selbst mitbringst und 150.000 € als Kredit aufnimmst, ist deine "Eigenkapitalquote" 50 %. Je höher, desto sicherer stehst du da.

**Formel:**
```
Eigenkapitalquote = Eigenkapital / Bilanzsumme × 100
```

**Beispiel:**
Eigenkapital = 40 Mio. €, Bilanzsumme = 100 Mio. €
→ **Eigenkapitalquote = 40 %**

**Faustregeln:**

| Eigenkapitalquote | Bewertung |
|---|---|
| > 40 % | 🟢 Sehr solide — Unternehmen ist gut finanziert |
| 25–40 % | 🟡 Ordentlich — branchenüblich für viele Sektoren |
| 15–25 % | 🟠 Eher dünn — genauer hinschauen |
| < 15 % | 🔴 Riskant — hohe Abhängigkeit von Fremdkapital |

> **Achtung:** Banken und Versicherungen haben naturgemäß sehr niedrige Eigenkapitalquoten (oft 5–10 %). Das liegt an ihrem Geschäftsmodell, nicht an schlechter Führung. Immer branchenintern vergleichen!

---

## 1.4 Verschuldungsgrad (Debt-to-Equity Ratio)

> **Was ist das?** Zeigt dir das Verhältnis von Schulden zu Eigenkapital. Wie viel Fremdkapital steht jedem Euro Eigenkapital gegenüber?

**Formel:**
```
Verschuldungsgrad = Fremdkapital / Eigenkapital
```

**Beispiel:**
Fremdkapital = 60 Mio. €, Eigenkapital = 40 Mio. €
→ **Verschuldungsgrad = 1,5** (= für jeden Euro EK gibt es 1,50 € Schulden)

**Faustregeln:**

| Verschuldungsgrad | Bewertung |
|---|---|
| < 0,5 | 🟢 Sehr konservativ finanziert |
| 0,5 – 1,0 | 🟡 Ausgewogen |
| 1,0 – 2,0 | 🟠 Erhöhte Verschuldung |
| > 2,0 | 🔴 Hoch verschuldet — Risiko steigt |

# 2. 💰 Die Gewinn- und Verlustrechnung (GuV) — Der Motor

> **Die Metapher:** Wenn die Bilanz ein **Foto** ist, dann ist die GuV ein **Film**. Sie zeigt dir nicht den Zustand, sondern die **Bewegung** — was über ein Jahr (oder Quartal) an Geld reinkam und was rausging.

```
  Umsatz (Revenue)
  − Herstellungskosten
  ─────────────────────────
  = Bruttogewinn (Gross Profit)
  − Vertriebs- & Verwaltungskosten
  ─────────────────────────
  = EBIT (operativer Gewinn)
  − Zinsen
  − Steuern
  ─────────────────────────
  = Nettogewinn (Net Income)
  ÷ Anzahl Aktien
  ─────────────────────────
  = Gewinn pro Aktie (EPS)
```

---

## 2.1 Umsatz (Revenue / Umsatzerlöse)

> **Was ist das?** Der Gesamtbetrag, den ein Unternehmen durch den Verkauf von Produkten oder Dienstleistungen einnimmt. Die **"Top Line"** — die oberste Zeile der GuV.

**Was sagt dir das?**
- Wie **groß** ist das Geschäft?
- Wächst oder schrumpft das Unternehmen?

> **Wichtig:** Umsatz allein sagt wenig über den Erfolg! Ein Unternehmen kann Milliarden umsetzen und trotzdem Verluste machen (siehe z. B. viele Tech-Startups).

---

## 2.2 Bruttogewinn und Bruttomarge

> **Was ist das?** Der Gewinn nach Abzug der **direkten Kosten** (Material, Produktion), aber BEVOR Verwaltung, Marketing und Co. bezahlt werden.

**Formel:**
```
Bruttomarge = Bruttogewinn / Umsatz × 100
```

**Beispiel:**
Umsatz = 100 Mio. €, Herstellungskosten = 40 Mio. €
→ Bruttogewinn = 60 Mio. €
→ **Bruttomarge = 60 %**

**Was verrät die Bruttomarge?**

| Bruttomarge | Was es bedeutet | Typische Branchen |
|---|---|---|
| > 70 % | 🟢 Sehr stark — das Unternehmen hat echte **Preismacht** | Software, Luxusgüter, Pharma |
| 40–70 % | 🟡 Gesund | Konsumgüter, Industrie |
| 20–40 % | 🟠 Dünn, aber branchenüblich | Einzelhandel, Lebensmittel |
| < 20 % | 🔴 Sehr eng — wenig Spielraum | Supermärkte, Rohstoffe |

> **Profi-Tipp:** Eine hohe Bruttomarge ist oft ein Zeichen für einen **Wettbewerbsvorteil** (Warren Buffetts "Moat"). Wenn ein Unternehmen seine Preise hoch halten kann, hat es etwas, das die Konkurrenz nicht einfach kopieren kann.

---

## 2.3 EBIT — Earnings Before Interest and Taxes

> **Was ist das?** Der **operative Gewinn** — was das Kerngeschäft verdient, bevor Zinsen und Steuern abgezogen werden.

**Warum ist EBIT wichtig?**
- Er zeigt dir, wie profitabel das eigentliche **Geschäft** ist
- Er ist unabhängig davon, wie das Unternehmen finanziert ist (viel/wenig Schulden)
- Er ist unabhängig vom Steuersatz des Landes
- Dadurch kannst du Unternehmen **international vergleichen**

---

## 2.4 EBITDA — Earnings Before Interest, Taxes, Depreciation and Amortization

> **Die Metapher:** EBITDA ist wie EBIT, aber noch "bereinigter". Stell dir vor, du vergleichst zwei Restaurants. Restaurant A hat sein Gebäude gerade neu gekauft (hohe Abschreibungen). Restaurant B hat ein altes, abgeschriebenes Gebäude. EBITDA **gleicht diesen Unterschied aus**, damit du die operative Leistung fair vergleichen kannst.

**Formel:**
```
EBITDA = EBIT + Abschreibungen + Amortisation
```

**Was sind Abschreibungen?**
Wenn ein Unternehmen eine Maschine für 1 Mio. € kauft und sie 10 Jahre nutzt, "schreibt" es jedes Jahr 100.000 € ab. Das ist ein buchhalterischer Aufwand — aber es fließt kein Geld! EBITDA rechnet diesen "Papieraufwand" wieder raus.

**EBITDA-Marge:**
```
EBITDA-Marge = EBITDA / Umsatz × 100
```

| EBITDA-Marge | Bewertung |
|---|---|
| > 30 % | 🟢 Exzellent |
| 20–30 % | 🟡 Gut |
| 10–20 % | 🟠 Ordentlich |
| < 10 % | 🔴 Schwach |

> **Kritik an EBITDA:** Warren Buffett mag EBITDA nicht besonders. Sein Argument: *"Glaubt das Management etwa, dass die Zahnfee die Investitionen bezahlt?"* Abschreibungen sind echte Kosten — Maschinen müssen irgendwann ersetzt werden. EBITDA ist nützlich zum Vergleichen, aber verlass dich nie allein darauf.

---

## 2.5 Nettogewinn (Net Income) und Gewinnmarge

> **Was ist das?** Das, was am Ende wirklich **übrig bleibt** — nach allen Kosten, Zinsen und Steuern. Die **"Bottom Line"**.

**Gewinnmarge (Nettomarge):**
```
Nettomarge = Nettogewinn / Umsatz × 100
```

**Beispiel:**
Umsatz = 100 Mio. €, Nettogewinn = 12 Mio. €
→ **Nettomarge = 12 %** (von jedem Euro Umsatz bleiben 12 Cent Gewinn)

| Nettomarge | Bewertung |
|---|---|
| > 20 % | 🟢 Herausragend |
| 10–20 % | 🟡 Sehr gut |
| 5–10 % | 🟠 Solide |
| < 5 % | 🔴 Eng — wenig Puffer |

---

## 2.6 Gewinn pro Aktie (EPS — Earnings Per Share)

> **Was ist das?** Der Nettogewinn aufgeteilt auf jede einzelne Aktie. DAS ist die Zahl, die **dich als Aktionär** direkt betrifft.

**Formel:**
```
EPS = Nettogewinn / Anzahl ausstehender Aktien
```

**Beispiel:**
Nettogewinn = 12 Mio. €, 4 Mio. Aktien im Umlauf
→ **EPS = 3,00 €** (jede Aktie "verdient" 3 €)

**Warum ist EPS so wichtig?**
- Es ist die Basis für das **KGV** (dazu kommen wir gleich!)
- Steigendes EPS über die Jahre = das Unternehmen wird **pro Aktie profitabler**
- Achte auf **verwässertes EPS** (diluted EPS) — das berücksichtigt Aktienoptionen für Mitarbeiter und zeigt ein realistischeres Bild

# 3. 🩸 Cashflow-Kennzahlen — Das Blut im Kreislauf

> **Die Metapher:** Gewinne können durch Buchhaltungstricks manipuliert werden. Aber **Cash lügt nicht**. Der Cashflow zeigt dir, wie viel echtes Geld durch die Adern des Unternehmens fließt. Ein Unternehmen kann Gewinne ausweisen und trotzdem pleite gehen — aber nie, wenn genug Cash fließt.

---

## 3.1 Operativer Cashflow (OCF)

> **Was ist das?** Das Geld, das tatsächlich aus dem normalen Geschäftsbetrieb hereinkommt — abzüglich dessen, was für den laufenden Betrieb rausgeht. Kein Kauf von Maschinen, keine Kredite — nur das Kerngeschäft.

**Was sagt dir das?**
- 🟢 **Positiv und steigend:** Das Kerngeschäft generiert echtes Geld — sehr gut!
- 🔴 **Negativ:** Das Unternehmen verbrennt Geld im Tagesgeschäft — problematisch (es sei denn, es wächst extrem schnell)

**Profi-Check: Ist der Gewinn "echt"?**
```
Wenn: Operativer Cashflow ≈ Nettogewinn → 🟢 Gewinne sind cashgestützt
Wenn: Operativer Cashflow << Nettogewinn → 🔴 Warnsignal! Gewinne könnten aufgeblasen sein
```

---

## 3.2 Free Cashflow (FCF) — Die Königskennzahl

> **Was ist das?** Der Cashflow, der übrig bleibt, **nachdem** das Unternehmen alles Nötige investiert hat (neue Maschinen, Gebäude, Software…). Das ist das Geld, das frei verfügbar ist — für Dividenden, Aktienrückkäufe oder Wachstum.

**Formel:**
```
Free Cashflow = Operativer Cashflow − Investitionsausgaben (CapEx)
```

**Warum ist FCF so wichtig?**
- FCF ist das, was **dir als Aktionär** gehört
- Unternehmen mit hohem FCF können Dividenden zahlen, Aktien zurückkaufen oder Schulden tilgen
- Viele Profis halten FCF für **aussagekräftiger als den Nettogewinn**

**FCF-Marge:**
```
FCF-Marge = Free Cashflow / Umsatz × 100
```

| FCF-Marge | Bewertung |
|---|---|
| > 20 % | 🟢 Cash-Maschine! Erstklassig |
| 10–20 % | 🟡 Sehr gut |
| 5–10 % | 🟠 Ordentlich |
| < 5 % oder negativ | 🔴 Wenig Spielraum oder Cash-Verbrennung |

---

## 3.3 Cashflow je Aktie (Cash Flow Per Share)

**Formel:**
```
Cashflow je Aktie = Operativer Cashflow / Anzahl Aktien
```

Analog zum EPS, aber auf Cash-Basis. Nützlich, um zu prüfen, ob hinter dem Gewinn pro Aktie auch echtes Geld steht.

# 4. ⚙️ Rentabilitätskennzahlen — Wie effizient arbeitet das Unternehmen?

> **Die Metapher:** Zwei Restaurants machen beide 1 Mio. € Gewinn. Restaurant A brauchte dafür 2 Mio. € Kapital, Restaurant B brauchte 10 Mio. €. Welches ist das bessere Geschäft? Natürlich A — es macht **mehr aus weniger**. Genau das messen Rentabilitätskennzahlen.

---

## 4.1 Eigenkapitalrendite (ROE — Return on Equity)

> **Was ist das?** Wie viel Rendite erwirtschaftet das Unternehmen auf das Geld, das den **Eigentümern** (dir!) gehört?

**Formel:**
```
ROE = Nettogewinn / Eigenkapital × 100
```

**Beispiel:**
Nettogewinn = 12 Mio. €, Eigenkapital = 40 Mio. €
→ **ROE = 30 %** → Für jeden Euro Eigenkapital erwirtschaftet das Unternehmen 30 Cent Gewinn

| ROE | Bewertung |
|---|---|
| > 20 % | 🟢 Hervorragend — Zeichen eines Qualitätsunternehmens |
| 15–20 % | 🟡 Sehr gut |
| 10–15 % | 🟠 Ordentlich |
| < 10 % | 🔴 Unterdurchschnittlich |

> **Achtung Falle:** Ein sehr hoher ROE (> 40 %) kann auch ein Warnsignal sein! Wenn das Eigenkapital sehr niedrig ist (z. B. durch hohe Schulden oder Aktienrückkäufe), wird der ROE künstlich nach oben verzerrt. Immer zusammen mit der **Eigenkapitalquote** betrachten!

---

## 4.2 Gesamtkapitalrendite (ROA — Return on Assets)

> **Was ist das?** Wie effizient nutzt das Unternehmen sein **gesamtes** Vermögen (Eigen- + Fremdkapital)?

**Formel:**
```
ROA = Nettogewinn / Bilanzsumme × 100
```

| ROA | Bewertung |
|---|---|
| > 10 % | 🟢 Sehr effizient |
| 5–10 % | 🟡 Gut |
| < 5 % | 🟠 Branchenabhängig prüfen |

**ROA vs. ROE:** ROA ist unbestechlicher als ROE, weil es die Verschuldung mitberücksichtigt. Wenn ROE hoch und ROA niedrig ist, arbeitet das Unternehmen vor allem mit **Fremdkapital** — höheres Risiko.

---

## 4.3 ROIC — Return on Invested Capital

> **Was ist das?** Die Rendite auf das **tatsächlich investierte Kapital** (Eigenkapital + zinstragendes Fremdkapital). Viele Profis halten ROIC für die **aussagekräftigste** Rentabilitätskennzahl.

**Formel:**
```
ROIC = NOPAT / Investiertes Kapital × 100

NOPAT = EBIT × (1 − Steuersatz)
Investiertes Kapital = Eigenkapital + zinstragende Schulden − überschüssiges Cash
```

**Warum ROIC?**
- ROIC > Kapitalkosten (WACC) = Das Unternehmen **schafft Wert** 💰
- ROIC < Kapitalkosten = Das Unternehmen **vernichtet Wert** 💸
- Ein dauerhaft hoher ROIC (> 15 %) ist das stärkste Zeichen für einen **Wettbewerbsvorteil**

# 5. 🏷️ Bewertungskennzahlen — Ist die Aktie ihren Preis wert?

> **Die Metapher:** Du stehst im Supermarkt. Zwei Flaschen Olivenöl: Eine kostet 5 €, die andere 15 €. Ist die teure automatisch besser? Nicht unbedingt! Aber wenn die teure von einem preisgekrönten Bio-Hersteller ist, aus erster Pressung stammt und dreimal so gut schmeckt — dann ist sie es vielleicht wert.
>
> Bewertungskennzahlen helfen dir zu entscheiden: **Bekomme ich für den Preis genug Qualität?**

---

## 5.1 ⭐ KGV — Kurs-Gewinn-Verhältnis (P/E Ratio)

> **Das ist DIE bekannteste Kennzahl an der Börse. Wenn du nur eine lernst, dann diese.**

**Was ist das?** Das KGV sagt dir: **Wie viele Jahre** müsste das Unternehmen den aktuellen Gewinn erwirtschaften, damit du deinen Kaufpreis zurückhast?

**Formel:**
```
KGV = Aktienkurs / Gewinn pro Aktie (EPS)
```

**Beispiel:**
Aktienkurs = 60 €, EPS = 3 €
→ **KGV = 20** → Du zahlst das 20-Fache des Jahresgewinns

**Was bedeutet das KGV?**

| KGV | Interpretation |
|---|---|
| < 10 | 🟢 Günstig — aber warum? Ist das Unternehmen in Schwierigkeiten? |
| 10–15 | 🟢 Fair bewertet — bei soliden Unternehmen ein guter Einstieg |
| 15–25 | 🟡 Normal — typisch für gute Unternehmen mit solidem Wachstum |
| 25–40 | 🟠 Hoch bewertet — hier wird viel Wachstum eingepreist |
| > 40 | 🔴 Sehr teuer — nur bei extrem hohem Wachstum gerechtfertigt |

> **Die drei KGV-Varianten:**
> - **Trailing KGV:** Basiert auf dem Gewinn der **letzten 12 Monate** (Vergangenheit)
> - **Forward KGV:** Basiert auf dem **erwarteten** Gewinn der nächsten 12 Monate (Zukunft)
> - **Shiller-KGV (CAPE):** Basiert auf dem **durchschnittlichen Gewinn der letzten 10 Jahre**, inflationsbereinigt. Glättet Schwankungen.

> **Wann trügt das KGV?**
> - Bei **Verlustjahren**: Kein positiver Gewinn = kein sinnvolles KGV
> - Bei **zyklischen Unternehmen**: In Boomjahren sieht das KGV niedrig aus — aber der Gewinn wird wieder sinken
> - Bei **starkem Wachstum**: Ein KGV von 50 kann bei 40 % Gewinnwachstum sogar günstig sein (→ PEG-Ratio)
> - Bei **einmaligen Sondereffekten**: Verkauf einer Tochtergesellschaft bläht den Gewinn einmalig auf

---

## 5.2 KUV — Kurs-Umsatz-Verhältnis (P/S Ratio)

> **Wann brauchst du das?** Wenn ein Unternehmen **noch keine Gewinne macht** (KGV nicht berechenbar), aber schon kräftig Umsatz erzielt. Typisch bei jungen Wachstumsunternehmen.

**Formel:**
```
KUV = Marktkapitalisierung / Jahresumsatz
```
oder
```
KUV = Aktienkurs / Umsatz pro Aktie
```

| KUV | Bewertung |
|---|---|
| < 1 | 🟢 Günstig — du bezahlst weniger als den Jahresumsatz |
| 1–3 | 🟡 Fair |
| 3–10 | 🟠 Teuer — starkes Wachstum nötig, um das zu rechtfertigen |
| > 10 | 🔴 Sehr teuer — nur bei extremem Wachstum (z. B. SaaS-Firmen) |

> **Schwäche:** KUV sagt nichts über **Profitabilität**. Ein Unternehmen mit 10 Mrd. € Umsatz aber 0 Gewinn hat trotzdem ein berechenbares KUV.

---

## 5.3 KBV — Kurs-Buchwert-Verhältnis (P/B Ratio)

> **Was ist das?** Vergleicht den **Börsenpreis** einer Aktie mit dem **Buchwert** (= Eigenkapital pro Aktie). Es beantwortet die Frage: *Was zahlst du im Verhältnis zu dem, was der Aktie laut Bilanz gehört?*

**Formel:**
```
KBV = Aktienkurs / Buchwert pro Aktie
```
wobei
```
Buchwert pro Aktie = Eigenkapital / Anzahl Aktien
```

| KBV | Interpretation |
|---|---|
| < 1 | 🟢 Du kaufst unter Substanzwert — das Unternehmen ist an der Börse weniger wert als sein Eigenkapital (Schnäppchen oder Warnsignal?) |
| 1–3 | 🟡 Normal |
| > 3 | 🟠 Hoher Aufschlag — der Markt erwartet starkes Wachstum oder hat einen immateriellen Wert (Marke, Patente) eingepreist |

> **Wann ist KBV besonders nützlich?**
> - Bei **Banken und Versicherungen** (deren Vermögenswerte größtenteils finanzielle Vermögenswerte sind)
> - Bei **Industrieunternehmen** mit viel physischem Vermögen
> - **Weniger nützlich** bei Tech-/Software-Unternehmen (deren wahrer Wert in Code, Talenten und Nutzern steckt — nicht in Gebäuden)

---

## 5.4 KCV — Kurs-Cashflow-Verhältnis (P/CF Ratio)

> **Was ist das?** Wie das KGV, aber statt des Gewinns wird der **operative Cashflow** verwendet. Vorteil: Cash ist schwerer zu manipulieren als Gewinne.

**Formel:**
```
KCV = Aktienkurs / Cashflow pro Aktie
```

| KCV | Bewertung |
|---|---|
| < 10 | 🟢 Günstig |
| 10–20 | 🟡 Fair |
| > 20 | 🟠 Teuer |

> **Profi-Tipp:** Vergleiche KCV mit KGV. Wenn KCV deutlich niedriger ist als KGV, verdient das Unternehmen mehr Cash als es an Gewinn ausweist — ein **positives Zeichen**.

---

## 5.5 EV/EBITDA — Enterprise Value to EBITDA

> **Was ist das?** DAS Werkzeug der Profis und Investmentbanker. Es vergleicht den **Gesamtwert des Unternehmens** (inklusive Schulden!) mit seiner operativen Ertragskraft.

**Formel:**
```
Enterprise Value (EV) = Marktkapitalisierung + Schulden − Cash

EV/EBITDA = Enterprise Value / EBITDA
```

**Warum ist EV/EBITDA besser als KGV?**
- Es berücksichtigt **Schulden** (ein Unternehmen mit 10 Mrd. Schulden ist weniger wert als eines ohne, selbst bei gleichem Gewinn)
- Es ist unabhängig von **Steuern und Zinsen** → bessere Vergleichbarkeit
- Es funktioniert auch bei **Unternehmen mit niedrigem oder negativem Gewinn**

| EV/EBITDA | Bewertung |
|---|---|
| < 8 | 🟢 Günstig |
| 8–12 | 🟡 Fair |
| 12–20 | 🟠 Teuer |
| > 20 | 🔴 Sehr teuer |

---

## 5.6 PEG-Ratio — Price/Earnings to Growth

> **Was ist das?** Das PEG-Ratio nimmt das KGV und setzt es ins Verhältnis zum **erwarteten Gewinnwachstum**. Es beantwortet die entscheidende Frage: *Ist das hohe KGV durch das Wachstum gerechtfertigt?*

**Formel:**
```
PEG = KGV / erwartetes jährliches Gewinnwachstum (in %)
```

**Beispiel:**
KGV = 30, erwartetes Gewinnwachstum = 30 % pro Jahr
→ **PEG = 1,0** → fair bewertet!

KGV = 30, erwartetes Gewinnwachstum = 10 %
→ **PEG = 3,0** → überbewertet!

| PEG | Interpretation |
|---|---|
| < 1 | 🟢 Unterbewertete Wachstumsaktie — potenzielles Schnäppchen! |
| 1 | 🟡 Fair bewertet im Verhältnis zum Wachstum |
| > 1 | 🟠 Teuer im Verhältnis zum Wachstum |
| > 2 | 🔴 Deutlich überbewertet |

> **Das ist die Lieblingskennzahl von Peter Lynch**, dem legendären Fondsmanager. Sein Motto: *"Ein KGV, das dem Wachstum entspricht, ist fair. Alles darunter ist ein Geschenk."*

---

# 6. 💎 Dividendenkennzahlen — Die Rendite des Geduldigen

> **Die Metapher:** Eine Dividende ist wie die Miete, die du bekommst, wenn du eine Wohnung vermietest. Dein Eigentum steigt vielleicht im Wert (Kursgewinn) — aber die Miete zahlt dir schon jetzt regelmäßig Geld aus.

---

## 6.1 Dividendenrendite (Dividend Yield)

**Formel:**
```
Dividendenrendite = Jährliche Dividende pro Aktie / Aktienkurs × 100
```

**Beispiel:**
Dividende = 2 € pro Aktie, Aktienkurs = 50 €
→ **Dividendenrendite = 4 %**

| Dividendenrendite | Bewertung |
|---|---|
| 0–1 % | Gering — Fokus liegt auf Wachstum, nicht Ausschüttung |
| 1–3 % | 🟡 Moderat — typisch für wachsende Qualitätsunternehmen |
| 3–5 % | 🟢 Attraktiv — guter Einkommensstrom |
| > 5 % | 🟠 Hoch — aber Vorsicht! Ist die Dividende nachhaltig? |
| > 8 % | 🔴 Warnsignal — der Kurs könnte gefallen sein (= "Value Trap") |

---

## 6.2 Ausschüttungsquote (Payout Ratio)

> **Was ist das?** Wie viel Prozent des Gewinns schüttet das Unternehmen als Dividende aus?

**Formel:**
```
Ausschüttungsquote = Dividende pro Aktie / Gewinn pro Aktie × 100
```

| Ausschüttungsquote | Bewertung |
|---|---|
| < 40 % | 🟢 Konservativ — viel Puffer, Dividende ist sicher |
| 40–60 % | 🟡 Gesund und ausgewogen |
| 60–80 % | 🟠 Hoch — wenig Spielraum für Investitionen |
| > 80 % | 🔴 Riskant — Kürzung der Dividende droht |
| > 100 % | 🚨 Unhaltbar! Das Unternehmen zahlt mehr aus, als es verdient |

---

## 6.3 Dividendenwachstum

**Wie viele Jahre** hat das Unternehmen die Dividende in Folge erhöht?

| Kategorie | Jahre | Bedeutung |
|---|---|---|
| **Dividend Aristocrats** | ≥ 25 Jahre | Elite — selbst in Krisen wurde nie gekürzt |
| **Dividend Champions** | ≥ 25 Jahre | Ähnlich, US-Fokus |
| **Dividend Achievers** | ≥ 10 Jahre | Sehr zuverlässig |
| **Neue Zahler** | < 10 Jahre | Noch nicht voll bewiesen |

> **Profi-Tipp:** Ein Unternehmen, das seit 25+ Jahren die Dividende erhöht (z. B. Johnson & Johnson, Procter & Gamble, Coca-Cola), wird die Dividende nur im absoluten Notfall kürzen. Das Management wird Berge versetzen, um den Aristocrat-Status zu behalten.

---

# 7. 🚀 Wachstumskennzahlen — Wohin geht die Reise?

> **Die Metapher:** Du kannst das beste Restaurant der Stadt haben — aber wenn jedes Jahr weniger Gäste kommen, ist es ein Auslaufmodell. Wachstum ist der Treibstoff für langfristige Kursgewinne.

---

## 7.1 Umsatzwachstum (Revenue Growth)

**Formel:**
```
Umsatzwachstum = (Umsatz aktuelles Jahr − Umsatz Vorjahr) / Umsatz Vorjahr × 100
```

| Umsatzwachstum | Bewertung |
|---|---|
| > 20 % | 🟢 Starkes Wachstum — typisch für Tech-/Growth-Aktien |
| 10–20 % | 🟡 Gesundes Wachstum |
| 5–10 % | 🟠 Moderates Wachstum |
| 0–5 % | ⚪ Geringes Wachstum — eher Value/Dividendenaktie |
| < 0 % | 🔴 Schrumpfendes Geschäft |

> **Wichtig:** Schaue dir **mindestens 3–5 Jahre** an, nicht nur ein einzelnes Jahr. Ein einmaliger Umsatzsprung (z. B. durch eine Übernahme) sagt wenig über die organische Kraft des Geschäfts.

---

## 7.2 Gewinnwachstum (EPS Growth)

**Formel:**
```
EPS-Wachstum = (EPS aktuell − EPS Vorjahr) / EPS Vorjahr × 100
```

**Noch wichtiger als Umsatzwachstum!** Denn:
- Ein Unternehmen kann den Umsatz steigern, aber trotzdem weniger verdienen (steigende Kosten)
- Langfristig folgt der Aktienkurs dem **Gewinnwachstum**

> **Die goldene Faustregel von Peter Lynch:** *"Der Aktienkurs folgt langfristig dem Gewinn pro Aktie wie ein Hund seinem Herrchen."*

---

## 7.3 Organisches vs. akquisitorisches Wachstum

| Art | Beschreibung | Bewertung |
|---|---|---|
| **Organisch** | Wachstum aus eigener Kraft — mehr Kunden, höhere Preise, neue Produkte | 🟢 Nachhaltig und wertvoll |
| **Akquisitorisch** | Wachstum durch Zukäufe anderer Unternehmen | 🟡 Kann gut oder schlecht sein — kommt auf den Preis an |

---

# 8. 🛡️ Verschuldung & Risiko — Wie sicher ist das Investment?

> **Die Metapher:** Schulden sind wie ein Hebel. Wenn alles gut läuft, vervielfachen sie den Gewinn. Wenn es schlecht läuft, vervielfachen sie die **Verluste**. In einer Krise gehen die Unternehmen unter, die zu viel Schulden haben.

---

## 8.1 Net Debt / EBITDA

> **Was ist das?** Wie viele Jahre bräuchte das Unternehmen, um seine Nettoverschuldung aus dem operativen Gewinn zurückzuzahlen?

**Formel:**
```
Net Debt = Gesamtschulden − Barmittel
Net Debt / EBITDA = Nettoverschuldung / EBITDA
```

| Net Debt / EBITDA | Bewertung |
|---|---|
| < 1 | 🟢 Sehr niedrig verschuldet |
| 1–2 | 🟡 Komfortabel |
| 2–3 | 🟠 Erhöht |
| > 3 | 🔴 Hoch — Risiko in Abschwüngen |
| > 5 | 🚨 Gefährlich |

---

## 8.2 Zinsdeckungsgrad (Interest Coverage Ratio)

> **Was ist das?** Kann das Unternehmen seine Zinszahlungen bequem aus dem operativen Gewinn bezahlen?

**Formel:**
```
Zinsdeckungsgrad = EBIT / Zinsaufwand
```

| Zinsdeckungsgrad | Bewertung |
|---|---|
| > 10 | 🟢 Zinsen sind kein Problem |
| 5–10 | 🟡 Komfortabel |
| 2–5 | 🟠 Wird eng |
| < 2 | 🔴 Gefahr — das Unternehmen kämpft mit seinen Zinsen |

---

## 8.3 Current Ratio (Liquidität 3. Grades)

> **Was ist das?** Kann das Unternehmen seine **kurzfristigen** Schulden (innerhalb 1 Jahr fällig) aus seinen kurzfristigen Vermögenswerten bezahlen?

**Formel:**
```
Current Ratio = Kurzfristige Vermögenswerte / Kurzfristige Verbindlichkeiten
```

| Current Ratio | Bewertung |
|---|---|
| > 2 | 🟢 Sehr liquide |
| 1,5–2 | 🟡 Gesund |
| 1–1,5 | 🟠 Eng — genauer hinschauen |
| < 1 | 🔴 Kurzfristige Zahlungsprobleme möglich |
